== Description ==

Heb je een eigen website met verwijzingen naar bijbelteksten, dan kun je daar nu popups bij krijgen met daarin het bijbelvers uit de BasisBijbel. Wanneer een bezoeker van je site met de muis op de tekstverwijzing gaat staan, verschijnt de popup.

Wilt u de ontwikkeling van de BasisBijbel WordPress plugin steunen? Maak dan een kleine donatie op http://stichting.basisbijbel.nl/doe_je_mee
